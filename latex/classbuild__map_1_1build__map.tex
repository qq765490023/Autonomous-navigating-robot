\doxysubsection{Detailed Description}
\begin{DoxyVerb}A class used to scan around and build topological map for autonomous navigation of the robot.
...
Attributes ---------- 

map_built: bool
When topological map is built this is set to true
   
can_build: bool
Set to true when markers are scanned

markers: list
Store total markers that have been received.

angle: float
The current angle of joint1(rotation of cameara) 

scanned: bool
Set to true if finishes scanning
   
Methods -------
room_scan():
This funtion will publish the desired angle of the cameara to move the robot to a desired angle.

Methods -------
build():
This funtion will get information of rooms from /room_info then build topological map\end{DoxyVerb}
